# Floating Dynamic Legend

*******************

The **Floating Dynamic Legend** tool allows users to view the legends of all active map layers in a dedicated panel.

!!! note
    The **Floating Dynamic Legend** plugin is not enabled by default. However, it can be configured within [application contexts](application-context.md#configure-plugins) if needed.

By clicking the **Floating Dynamic Legend** <img src="../img/button/dynamic-legend-button.jpg" class="ms-docbutton"/> button available in the [Side Toolbar](mapstore-toolbars.md#side-toolbar), users can view the legends for all layers present on the map.

<img src="../img/floating-dynamic-legend/floating-dynamic-legend-panel.jpg" class="ms-docimage"/>

The legend displayed by the Floating Dynamic Legend works in the same way as the "Dynamic Legend" option, which can be enabled for each layer from the TOC as described in the [Layer Settings](layer-settings.md#display) section.
