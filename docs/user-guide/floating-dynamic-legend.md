# Floating Dynamic Legend

*******************

The **Floating Dynamic Legend** tool allows users to view the legends of all active map layers in a dedicated panel.

!!! note
    The **Floating Dynamic Legend** plugin is not enabled by default. However, it can be configured within [application contexts](application-context.md#configure-plugins) using the desired provider.

By clicking the **Floating Dynamic Legend** <img src="../img/button/dynamic-legend-button.jpg" class="ms-docbutton"/> button available in the [Side Toolbar](mapstore-toolbars.md#side-toolbar). Once activated, it allows the user to view the legends of the layers present on the map.

<img src="../img/floating-dynamic-legend/floating-dynamic-legend-panel.jpg" class="ms-docimage"/>

The legend is *dynamic* because it automatically filters and updates its content based on what is currently visible on the screen. The legend will adjust in real-time when you:

- **Zoom in or out**

- **Pan** across the map

- **Toggle** layer visibility on or off
